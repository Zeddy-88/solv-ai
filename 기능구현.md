## 1. 사이드바 (Sidebar)

### 1.1 구성 요소

| 영역 | 구현 내용 |
|---|---|
| 로고 영역 | `w-8 h-8` Klein Blue 배경, Lucide `zap` 아이콘 (fill-white), "Solv AI" 텍스트 |
| 새로운 재무분석 버튼 | `bg-klein`, `rounded-xl`, full-width, Lucide `plus` 아이콘, `hover:opacity-90` |
| 검색 버튼 | `text-gray-500`, Lucide `search` 아이콘, `hover:bg-white` |
| 구분선 | `h-px bg-black/10 mx-5` |
| 즐겨찾기 섹션 헤더 | `text-[12px] font-bold text-gray-400 uppercase tracking-widest` |
| 즐겨찾기 항목 | 회사 이니셜 아바타 + 회사명 + `star` 아이콘 (fill-[#EF9F27]) |
| 이력 목록 헤더 | "최근 분석 이력" 레이블 |
| 이력 항목 | 색상 점(●) + 회사명 + 날짜("오늘") — `flex-1 overflow-y-auto` |
| 사용자 프로필 | `border-t`, 원형 아바타(Klein Blue) + 이름 + settings 아이콘 |

### 1.2 모바일 동작

```javascript
// 사이드바 토글 (모바일 전용)
function toggleSidebar() {
  const sidebar = document.getElementById('sidebar');
  const overlay = document.getElementById('sidebarOverlay');
  const isHidden = sidebar.classList.contains('-translate-x-full');
  
  if (isHidden) {
    // 열기
    sidebar.classList.remove('-translate-x-full');
    overlay.classList.remove('hidden');
  } else {
    // 닫기
    sidebar.classList.add('-translate-x-full');
    overlay.classList.add('hidden');
  }
}
// 오버레이 클릭 시 닫힘: onclick="toggleSidebar()"
```

### 1.3 이력 항목 색상 점 의미

| 색상 | 의미 | 상태 |
|---|---|---|
| `bg-[#378ADD]` (파란색) | 현재 선택/열람 중 | 활성 |
| `bg-[#639922]` (초록색) | 이전 분석 완료 | 일반 |
| `bg-yellow-400` | 즐겨찾기 | 즐겨찾기 |

### 11.2 아코디언 렌더링

```javascript
function renderScripts() {
  document.getElementById('scriptContainer').innerHTML = scripts.map(s => `
    <div class="bg-white rounded-3xl border border-black/10 shadow-md 
                overflow-hidden transition-all hover:shadow-xl">
      
      <!-- 헤더: 클릭 시 펼침 -->
      <div class="p-6 flex items-center justify-between cursor-pointer" 
           onclick="toggleScript(${s.id})">
        <div class="flex items-center gap-5">
          <!-- 번호 원형 -->
          <div class="w-8 h-8 rounded-full bg-gray-100 text-gray-600 
                      flex items-center justify-center text-sm font-black shadow-inner">
            ${s.id}
          </div>
          <span class="text-lg font-black text-gray-900">${s.type}</span>
          <span class="px-4 py-1 bg-blue-50 text-blue-700 text-xs rounded-full font-black">
            ${s.tag}
          </span>
        </div>
        <!-- 쉐브론 아이콘 (펼침 시 180도 회전) -->
        <i data-lucide="chevron-down" id="chevron-${s.id}" 
           class="w-6 h-6 text-gray-400 transition-transform"></i>
      </div>
      
      <!-- 컨텐츠 (기본: hidden) -->
      <div class="px-10 md:px-16 pb-8 hidden animate-fade" id="script-content-${s.id}">
        <!-- 스크립트 텍스트: font-normal, text-[15px] (UI/UX 스펙) -->
        <div class="bg-gray-50 p-5 rounded-2xl text-[15px] leading-relaxed 
                    border border-gray-100 mb-6 font-normal text-gray-800">
          ${s.script}
        </div>
        
        <!-- WWH 카드 3열 그리드 -->
        <div class="grid grid-cols-1 sm:grid-cols-3 gap-4">
          ${['What', 'Why', 'How'].map(label => `
            <div class="p-4 bg-white rounded-xl border border-black/5 shadow-sm">
              <p class="text-[11px] font-black uppercase mb-1.5 tracking-widest 
                        ${label==='What'?'text-blue-500':label==='Why'?'text-green-500':'text-amber-500'}">
                ${label}
              </p>
              <p class="text-xs text-gray-600 leading-relaxed font-bold">
                ${s.wwh[label.toLowerCase()]}
              </p>
            </div>
          `).join('')}
        </div>
      </div>
    </div>
  `).join('');
  lucide.createIcons();
}
```

### 11.3 스크립트 토글 함수

```javascript
function toggleScript(id) {
  const content = document.getElementById(`script-content-${id}`);
  const chevron = document.getElementById(`chevron-${id}`);
  content.classList.toggle('hidden');
  chevron.style.transform = content.classList.contains('hidden') 
    ? 'rotate(0deg)' 
    : 'rotate(180deg)';
}
```
